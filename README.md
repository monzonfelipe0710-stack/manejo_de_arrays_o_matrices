# Manejo de Arrays o Matrices 📊

Este repositorio contiene la solución a un ejercicio de lógica de programación utilizando estructuras de datos tipo matriz (arrays bidimensionales). El proyecto está desarrollado tanto en **Python** como en **JavaScript (Node.js)** y se ejecuta íntegramente a través de la terminal (línea de comandos).

## 🚀 Funcionalidades

El sistema permite al usuario ingresar datos de múltiples alumnos de forma continua y realiza los siguientes procesos:
1. **Registro de datos:** Solicita Nombre, Edad y Nota, almacenándolos en una matriz con el formato `[nombre, edad, nota]`.
2. **Corte dinámico:** El ingreso de datos se detiene al escribir la palabra `finalizar` o `terminado`.
3. **Listado Original:** Imprime en consola todos los registros en el orden exacto en el que fueron cargados.
4. **Ordenamiento:** Imprime un segundo listado de los alumnos, pero ordenado por su nota de **mayor a menor**.
5. **Cálculo de Promedio:** Calcula y muestra el promedio general de las notas de todos los alumnos registrados.

## 🛠️ Tecnologías utilizadas

* **Python 3**
* **JavaScript** (Entorno Node.js)
* **Git / GitHub** (Flujo de trabajo con ramas `main` y `develop`)
